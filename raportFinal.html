Detalii despre paginile HTML implementate

Pagina principala

Prima pagina a site-ului permite unui utilizator acces la alte 5 pagini: pagina de invatare HTML la care se ajunge prin apasararea butonului "INVATA" aflat in sectiunea specifica HTML ului, pagina de exersare a exercitiilor HTML, pagina de invatare CSS,pagina de exercitii CSS si nu in ultimul rand icon ul alfat in partea din stanga sus permite utlizatorului accesul pe pagina de logare.

Pagina de logare

Odata ajuns pe pagina de logare utilizatorul poate intra in contul sau folosindu-se de numele de utilizator si parola.Daca nu dispune de asa ceva acesta poate alege sa se logheze cu ajotorul contului de Github permitand accesul aplicatiei la datele sale publice din Github. Daca insa acesta nu are cont de Github se poate inregistra in baza de date a aplicatiei. Din pagina de logare utilizatorul poate ajunge din nou pe pagina principala,sau pe paginile de invatare HTML si CSS.

Pagina de inregistrare

Utilizatorul isi poate crea un cont folosindu-se de numele si prenumele sau,alaturi de un nume de utilizator, o adresa de email si o parola.Unele elemente introduse trebuie sa fie unice dar utilizatorul va fi anuntat de aplicatie daca acestea nu sunt. Din aceasta pagina user-ul se poate intoarce pe pagina de Logare,poate vizita paginile de invatare.

Pagina de invatare HTML

Daca utilizatorul ajunge pe aceasta pagina acesta poate sa aleaga diferite topici, legate de HTML pe care sa le studieze. Acesta se poate intoarce pe pagina principala sau daca este logat isi poate vizita pagina de profil,cea de statistici. De asemenea daca un utilizator nelogat ajunge pe aceasta pagina se poate loga, si poate vedea clasamentul.

Pagina de invatare CSS

Daca utilizatorul ajunge pe aceasta pagina acesta poate sa aleaga diferite topici, legate de HTML pe care sa le studieze. Acesta se poate intoarce pe pagina principala sau daca este logat isi poate vizita pagina de profil,cea de statistici. De asemenea daca un utilizator nelogat ajunge pe aceasta pagina se poate loga, si poate vedea clasamentul.

Pagina de Profil

Pe aceasta pagina user-ul isi poate vedea datele din baza de date:numele de utilizator,numele,prenumele id-ul din baza de date si email-ul.Acesta poate alege sa editeze unele informatii apasand butonul Edit. Din aceasta pagina poate ajunge fie acasa,fie se poate deloga sau ajunge pe pagina de Editare a informatiilor.

Pagina de statistici

Aici userul vede cate zile consecutive a petrecut pe aplicatie.

Pagina de alegere a exercitiilor CSS

Utilizatorul poate alege nivelul de dificulate al intrebarilor.

Pagina de Game

Spre deosebire de prima noastra prezentare, jocul nu se mai desfasoara acum ca fiind static si intrabarile sa fie puse direct din HTML-ul aplicatiei,de aceasta data facem call-uri la un API de unde sunt luate intrebarile in functie de nivelul de dificulate. Pagina de game are un buton de hint care este un link spre paginile de invatare,un buton de quit in cazul in care user-ul vrea sa abandoneze, un buton de logout.De asemenea,pe masura ce raspunde corect la intrebari scorul acestuia creste,la fel si bara de progres care arata la cate intrebari a ramas. Cand termina nivelul,ajunge la numarul maxim de intrebari permise,user-ul este redirectionat pe pagina de sfarsit.

Pagina de sfarsit

Utilizatorul poate alege sa salveze scorul in clasament,sa joace din nou,sau sa mearga pe pagina principala.

Design

In ceea ce priveste design-ul am ales sa renuntam la imaginile ce sa aflau pe paginile de invatare,dar le am pastrat-o pe cea de pe pagina de login si Register.Am optat pentru a alege ca si culoare dominanta movul,iar pe pagina prinicala se poate observa ca am ales sa facem deosebire intre sectiunile de HTML si CSS folosind culorile lor definitoare. In ceea ce priveste design ul jocului am ales sa mergem pe o abordare putin diferita, toate intrebarile sunt de tip grila cu 3 varinate de raspuns si se bazeaza pe aceeasi tema cromatica ca si pagina principala.
